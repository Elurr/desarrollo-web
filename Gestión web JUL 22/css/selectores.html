Selectores de CSS

Los selectores son grupos de elementos sobre los que aplicamos ciertas reglas de CSS.

Selectores generales

Agrupan a todos los elementos cuya etiqueta coincida con el nombre del selector 

                p {
                    color: red;
                }

Clases e IDs

Las clases son grupos de elementos a los que se les puede aplicar las mismas reglas de CSS. 

                .grupo1 {
                    font-weight: 500;
                    background-color: blue;
                }

Las IDs identifican a un único elemento, por lo que las reglas de CSS sólo se aplicarán a éste. 

                #elem1 {
                    font-weight: 300;
                    color: white;
                }

Selector universal

Agrupa a todos los elementos HTML del documento 

                * {
                    color: red;
                }

Selectores avanzados

Seleccionar elementos dentro de una clase

Se pueden combinar selectores de clase y generales para distinguir elementos dentro de una clase. 

                p.grupo1 {
                    text-align: right;
                }

Agrupando selectores

Si vamos a usar las mismas reglas para distintos tipos de elementos, los selectores se pueden agrupar. 

                h1 {
                    text-align: center;
                }
                h2 {
                    text-align: center;
                }
                span {
                    text-align: center;
                }

Es lo mismo que:

                h1, h2, span {
                    text-align: center;
                }

Combinadores

Un selector puede contener uno de los siguientes 4 comibnadores:

  • Descendiente (espacio)
  • Hijo directo (>)
  • Hermano adyacente (+)
  • Hermano general (~) (se escribe con Alt+126)
