Tipos de datos en JavaScript

Tipos

  • strings: Cadenas de caracteres
  • number o int: Números operables
  • boolean: valores binarios (true, false o 0, 1)
  • arrays: cadenas de valores
  • objetos: cadenas de valores con propiedades

Números

JavaScript sólo tiene un tipo de número. Éstos pueden escribirse con o sin decimales.

Los números de JS son siempre float de doble presición (double). Son precisos hasta 15 dígitos.

Arrays

Un array es un tipo especial de variable. Puede contener más de un valor.

Los arrays son útiles cuando queremos trabajar con listas de datos de larga longitud.

Objetos

Un objeto, al igual que los arrays, puede contener numerosos valores. La principal diferencia es que el índice de los objetos tiene nombre.

Los arrays son objetos especiales cuyo índice está numerado.

Ejemplo

Un objeto coche tiene propiedades como peso y color. Y también tiene métodos como arrancar y parar.

Todos los objetos de tipo coche comparten las mismas propiedades aunque sus valores varían.

Todos los coches tienen los mismos métodos pero varía su ejecución.
