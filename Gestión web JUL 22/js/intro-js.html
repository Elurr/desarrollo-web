Variables

Las variables son bloques de memoria donde almacenamos información.

En javascript hay varios tipos de variables, las más básicas son:

  • strings o cadenas de texto.
  • ints o valores numéricos.
  • booleanos con valores binarios.

Declarar variables en js

Según el alcance o "scope" que queramos que una variable tenga, las vamos declarar con los siguientes "statements".

  • var o declaracion global o genérica.
  • let o declaracion local.
  • const cuando queramos declarar una variable cuyo valor es estático.

Bucles

Hay distintos tipos de bucles en JavaScript. Éstos son bloques de código que se ejecutan un número repetido de veces.

  • for: se ejecutan x número de veces, siendo x configurable por nosotros.
  • while: similar a los for, se ejecutan mientras una condición se cumpla (true)
  • do while: mientras que while comprueba la condición antes de ejecutarse, éste se ejecuta una vez y luego comprueba la condición.

Operadores de asignación

Estos operadores asignan valores a las variables de JS

  • x += y (x = x + y)
  • x -= y (x = x - y)
  • x *= y (x = x * y)
  • x /= y (x = x / y)
  • x %= y (x = x % y)
  • x **= y (x = x ** y)

Operadores de comparación

Se usan en los condicionales y comparan dos valores o tipos

  • == equivale a (x == 5: true)("5" == x: true)
  • === mismo valor y tipo (x === 5: true)(x === "5": false)
  • != no equivale a
  • !== distinto valor o tipo
  • > mayor que
  • < menor que
  • >= mayor que o igual a
  • <= menor que o igual a

Operadores lógicos

Estos operadores se usan en conjunto con los operadores de comparación

Se usan para determinar la lógica entre variables o valores

  • && (and)
  • || (or)
  • ! (not)
  • ? (operador ternario)
