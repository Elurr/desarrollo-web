Elementos avanzados de HTML

Links o hipervínculos

Los links se encuentran en todo tipo de páginas. Permiten al usuario navegar por los documentos de la web.

Cuando clicas en un link te mueve a otro documento o a una parte concreta del mismo.

Cuando se pasa el cursor sobre un link, éste se torna en una mano.

Un link se define mediante el elemento <a>

La sintaxis de un link, por norma general, es:
<a href="url">Texto del link</a>

El atributo más importante de un link es el "href" ya que es éste el que indica el punto al que nos llevará el link.

Ejemplos:

Link a Google
Link a la teoría de CSS

Los links tienen 3 estados:

  • Son azules cuando no han sido visitados.
  • Morados cuando ya han sido clicados.
  • y rojos cuando están activos.

Los links se pueden modificar mediante CSS para que tengan un aspecto completamente distinto.
